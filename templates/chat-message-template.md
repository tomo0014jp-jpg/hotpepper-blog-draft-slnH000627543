# Google Chat 送信フォーマット

Webhook へは合計4回 POST する。順序固定。
各 POST のボディは `{ "text": "<下記テンプレートに従った本文>" }`。

## 1通目：見出しメッセージ


{MODE_TAG}ブログ下書き {YYYY-MM-DD}｜{店舗名} 3本分
本日の3テーマ：
{テーマ1のタイトル}
{テーマ2のタイトル}
{テーマ3のタイトル}
このあと、記事本文を3メッセージに分けて投稿します。

- `{MODE_TAG}` は `config/run-mode.md` の値に従い、`【TEST】` または `【PROD】` を入れる。
- PROD で PR 運用の場合は、最終行に
  「※ past-topics.md は PR で更新済み。翌日実行前にマージしてください。」を追記する。

## 2〜4通目：記事メッセージ（記事1, 記事2, 記事3）


{MODE_TAG}記事{N}/3：{タイトル}
{本文 600〜1000文字}
ハッシュタグ：{#タグ1 #タグ2 …}

## 整形ルール
- 絵文字は使用しない
- Google Chat の簡易マークダウンのみ使用：`*太字*`、`_斜体_`、`` `code` ``
- 1メッセージは 4,000 文字を超えないこと
- 改行は通常の `\n`（JSON ペイロードでは `\n` としてエスケープ）
- HTML タグは使わない
