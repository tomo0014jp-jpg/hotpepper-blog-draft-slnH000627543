# test-topics.md — TEST用の生成テーマ履歴

> **このファイルは TEST モードで生成したテーマのみを記録します。**
> PROD モードで生成したテーマはここに追記しないでください（`data/past-topics.md` に記録）。
> ここに記録されたテーマは、本番（PROD）の重複判定の対象には**含まれません**。

## 使い方

- TEST モード実行時、Claude はこのファイルを読み、ここに記録済みのテーマと重複しないようにブログ3本を生成します。
- 生成後、Claude は新しい3本のテーマをこのファイルの末尾に追記します。
- TEST → PROD 切り替え時、このファイルの内容は **PROD の重複判定には引き継がれません**（意図通り）。

## 記録フォーマット

各エントリは以下の形式で追記してください。

```
## YYYY-MM-DD（TEST）
- 記事1: タイトル ／ 一言要旨
- 記事2: タイトル ／ 一言要旨
- 記事3: タイトル ／ 一言要旨
```
